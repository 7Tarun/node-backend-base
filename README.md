## Run Locally

Clone the project

```bash

```

Go to the project directory

```bash
  cd Suggee-Backend
```

Install dependencies

```bash
  npm install
```

Start the server dev mode

```bash
  npm run Suggee-Backend
```

Start the server Production

```bash
  npm run build
```

```bash
  npm run start
```
